# 🧠 NeoLearn – Plateforme d’apprentissage IT tout-en-un

NeoLearn est une plateforme d’e-learning innovante développée avec le **MERN stack (MongoDB, Express.js, React.js, Node.js)**. Elle vise à guider les apprenants à travers des parcours structurés dans divers domaines de l'informatique tels que le développement web, mobile, l’IA, le design, la cybersécurité, et plus encore.

## 🚀 Fonctionnalités principales

- 🎓 **Parcours d’apprentissage personnalisés** (frontend, backend, full stack, UI/UX, etc.)
- 📚 **Ressources variées** : vidéos, livres PDF, mindmaps (liens directs vers les sources)
- 📝 **Section Blog** : créer des articles, commenter, aimer, répondre
- 💬 **Communautés** par domaine : discussions en temps réel, entraide
- 📥 **Formulaire de contact** pour contacter l’administrateur
- 👤 **Espace utilisateur** : inscription, connexion, mot de passe oublié, édition du profil
- 🧠 **Améliorations futures** : intégration d'une **IA personnalisée** pour accompagner l’apprentissage

## 🛠️ Stack technique

- **Frontend** : React.js + Tailwind CSS
- **Backend** : Node.js + Express.js
- **Base de données** : MongoDB
- **Authentification** : JWT (JSON Web Token)
- **Communication en temps réel** : Socket.IO (pour les communautés)
- **Déploiement** : *à personnaliser selon GitHub Pages, Vercel, Render, etc.*
